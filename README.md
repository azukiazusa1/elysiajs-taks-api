# ElysiaJS Task API

A simple task API built with [ElysiaJS](https://elysiajs.com/).

## Requirements

- Bun

## Installation

Install Bun:

```bash
curl https://bun.sh/install | bash
```

Install dependencies:

```bash
bun install
```

Create Database Table:

```bash
bun run create-table
```

## Development
To start the development server run:
```bash
bun run dev
```

## Testing

```bash
bun test
```
